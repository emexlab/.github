## Welcome to emexLabs 👋
## emexOS
[emexOS](https://github.com/emexlab/emexOS) is a graphical operating system entirely written from scratch in C by emex and [all its contributors](https://github.com/emexlab/emexOS/graphs/contributors). It's designed for developers and enthusiasts, who want to customize their system and want full control over their systems.

## emexDE
[emexDE](https://github.com/emexlab/emexDE) is an iOS app that empowers developers with a full toolchain they can use even while being offline for iOS development on iPhone. It supports Swift, C, Objective-C, C++ and Objective-C++. It’s a powerful Xcode alternative that made the impossible possible, a fully on-device iOS IDE that doesn’t even need a cloud and can be used with airplane mode enabled. It officially supports iOS 16 all the way up to the latest iOS version. You can compile and run iOS apps on the go with ease, using the entire iOS 26.5 SDK.

## emex64
[emex64](https://github.com/emexlab/emex64) is a 64bit lightweight architecture. It's a mix of RISC and CISC and is not based on any previous architecture, it's only inspired by architectures like aarch64.
